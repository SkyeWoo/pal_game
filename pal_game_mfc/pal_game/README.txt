使用说明:

1. 工程是在vs2015环境下完成的, 请先新建Windows应用程序项目, 将.cpp .h文件导入到工程相应位置.

2. 将pal_game.rc文件导入到资源文件中, 保证images文件夹和代码在同一目录下.

3. 运行工程.

游戏说明:

1. 开始界面下可直接Enter键进入游戏, 也可按I键查看说明, ←→键控制翻页, Esc键退出.

2. ←↑→↓键操控角色移动, 空格键发射子弹攻击敌方.

3. 躲避敌方暗器, 避免与敌方相撞.

4. 游戏过程中会降落对角色有利的物品, 注意性别和专属武器的区别, 拿到不符合角色的物品视为无效.

5. 若角色死亡按Enter键切换下一角色, 或在游戏中直接按R结束当前角色, 切换下一角色.

6. 若所有角色都死亡, 游戏失败.

7. 达到相应分数, 游戏胜利.

8. 过关分数, 玩家血量、攻击、速度可在common.h中设定.
