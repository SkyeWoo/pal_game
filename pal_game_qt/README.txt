使用说明:

1. 工程是在Qt Creator4.2.0环境下完成的, 请将代码文件夹解压到不含中文字符的目录下.

2. images和snd文件夹是资源需要, 与代码放在同一目录下即可.

3. 打开pal_game.pro, 构建工程.

4. 运行工程.

快捷键说明:

1. 开始界面下, C设置音效, Enter开始游戏, I查看说明, Q退出游戏.

2. 音效设计面板下, ↑↓键选择, Y确定, N取消.

3. 说明面板下, ←→键翻页, Esc退出说明面板.

4. 游戏界面下, ←↑→↓键操控角色移动, 空格键发射子弹攻击敌方. R结束当前角色, 切换下一角色.

5. 游戏过程中Esc可暂停游戏, 暂停情况下Y继续游戏, N结束游戏并回到开始界面.

6. 游戏结束(胜利或失败)Enter返回开始界面.

鼠标控制说明:

1. 因为对角色进行了速度设计, 所以不方便用鼠标控制, 只能用←↑→↓键操控角色移动.

2. 除游戏运行过程中, 其他所有控件选择都可用鼠标完成.

游戏说明:

1. 每个玩家有一定的血量，攻击，防御，速度。血量即HP，决定角色能存活多久。攻击越高子弹及自身对敌军攻击也越大。防御越高受到敌军伤害越小。速度越高角色移动越快。

2. 击杀敌方以获取相应分数, 最终目标是达到一定分数获得游戏胜利.

3. 躲避敌方暗器, 避免与敌方相撞. 当然也可通过相撞击杀敌方获得分数.

4. 游戏过程中会降落对角色有利的物品, 注意性别和专属武器的区别, 拿到不符合角色的物品视为无效.

5. 若角色死亡游戏会进入暂停状态, 此时继续游戏会切换到下一角色.

6. 若所有角色都死亡, 游戏失败. 达到预设分数, 游戏胜利. 

7.过关分数, 玩家血量、攻击、速度可在common.h中设定.
